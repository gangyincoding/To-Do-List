## 📱 第一部分：科大讯飞开放平台注册

### 1.1 注册地址
🔗 **官网**：https://www.xfyun.cn/

### 1.2 注册步骤详解

#### Step 1: 账号注册（5分钟）

1. **访问官网**
   ```
   https://www.xfyun.cn/
   ```

2. **点击注册**
   - 位于页面右上角
   - 选择"个人注册"

3. **填写注册信息**
   ```
   手机号：[您的手机号]
   验证码：[接收短信验证码]
   密码：[设置强密码，建议8位以上，包含大小写字母和数字]
   ```

4. **完成注册**
   - 点击"注册"按钮
   - 记录账号信息

#### Step 2: 实名认证（15-30分钟）

1. **登录账号**
   - 登录后点击右上角头像
   - 选择"控制台"

2. **开始实名认证**
   - 首次进入会自动提示实名认证
   - 或手动点击"实名认证"

3. **选择认证方式**
   ```
   推荐选择：个人认证（速度快，适合个人开发者）
   企业认证：需要营业执照材料
   ```

4. **上传身份证信息**
   ```
   - 身份证正面照片（确保清晰，无反光）
   - 身份证反面照片（确保清晰，无反光）
   - 人脸识别验证（在光线充足的环境下进行）
   ```

5. **等待审核**
   - 通常5-10分钟完成
   - 审核通过后会收到短信通知

#### Step 3: 创建应用（5分钟）

1. **进入控制台**
   ```
   左侧菜单 → 我的应用 → 创建应用
   ```

2. **填写应用信息**
   ```
   应用名称：TaskAI智能待办
   应用分类：工具类应用
   应用描述：基于语音识别和AI大模型的智能待办清单应用
   ```

3. **选择服务能力**
   ```
   ✅ 实时语音转写（大模型版）
   ✅ 语音听流（可选）
   ❌ 语音合成（暂时不需要）
   ```

4. **提交申请**
   - 点击"创建应用"
   - 等待系统审核（通常几分钟）

#### Step 4: 获取API密钥（3分钟）

1. **进入应用详情**
   - 在"我的应用"中找到刚创建的应用
   - 点击应用名称进入详情

2. **查看API密钥**
   在应用详情页面您会看到：
   ```
   APPID：832d5dde（示例）
   APISecret：NGExZjIwY2E3YmNmODlmOGMwNGQ0Njgz（示例）
   APIKey：8413aab881f2294c5b7358db27067c15（示例）
   ```

3. **保存密钥信息**
   - 建议截图保存
   - 或复制到安全的记事本中

### 1.3 重要信息记录

科大讯飞API信息：
```
官网：https://www.xfyun.cn/
控制台：https://console.xfyun.cn/
API服务：实时语音转写（大模型版）

密钥信息：
- APPID：[您的APPID]
- APISecret：[您的APISecret]
- APIKey：[您的APIKey]
WebSocket地址：wss://office-api-ast-dx.iflyaisol.com/
```

### 1.4 常见问题解决

#### 问题1：实名认证失败
**解决方法**：
- 确保身份证照片清晰，无反光和遮挡
- 在光线充足的环境下进行人脸识别
- 使用Chrome浏览器，避免使用IE

#### 问题2：创建应用被拒
**解决方法**：
- 应用名称不要包含敏感词汇
- 应用描述要详细具体
- 选择正确的应用分类

#### 问题3：找不到API密钥
**解决方法**：
- 确保应用已创建成功
- 在应用详情页查找"API密钥"标签
- 如果找不到，可以创建新应用
