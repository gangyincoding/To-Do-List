# TaskAI 技术验证 Demo

## 📋 项目说明
这是TaskAI项目的技术验证Demo，用于测试语音识别和AI任务拆分的可行性。

## 🚀 快速开始

### 1. 打开Demo页面
直接用浏览器打开 `index.html` 文件即可。

### 2. 当前功能状态
- ✅ 基础UI界面完成
- ✅ 录音按钮交互完成
- ✅ 模拟语音识别功能
- ✅ 模拟AI任务拆分功能
- ❌ 真实API集成（待完成）

### 3. 测试流程
1. 点击"点击录音"按钮
2. 系统会模拟语音识别过程（3秒）
3. 自动调用AI拆分功能
4. 显示拆分后的任务列表
