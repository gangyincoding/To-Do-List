## 🔐 安全提醒

⚠️ **重要提醒**：
- `API.md` 文件包含真实的API密钥，请勿泄露或提交到公开仓库
- 在生产环境中，请使用环境变量或安全存储方案管理API密钥
- 定期更新API密钥以确保安全性

## 🚀 快速开始

1. 首先阅读 `API_REGISTRATION_COMPLETE_GUIDE.md` 了解注册流程
2. 按照指南注册相关API服务
3. 在 `API.md` 中配置您的API密钥
4. 参考 `BAIDU_API_SETUP_GUIDE.md` 进行百度API配置
