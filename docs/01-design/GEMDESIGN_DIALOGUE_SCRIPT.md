# GemDesign 对话脚本 - TaskAI 移动端原型设计

## 🎯 对话目标
使用 https://design.gemcoder.com/book/blog/ 与GemDesign AI对话，设计出TaskAI移动端应用的原型图。

## 📱 项目背景信息

### 🎯 产品名称
**TaskAI - 智能语音待办清单**

### 📝 产品定位
基于语音识别和AI大模型的智能待办清单应用，用户可以通过语音输入任务描述，系统自动将语音转换为文字，并使用AI智能拆分为具体的可执行任务列表。

### 🎯 核心价值主张
"说话即添加，智能即整理" - 语音输入 + AI任务拆分

### 👥 目标用户
- 习惯使用待办清单的人群
- 移动端为主要使用场景
- 早晨规划任务、工作间隙记录的使用场景

## 🎨 设计要求

### 📱 应用类型
- **移动端应用**：iOS和Android
- **屏幕尺寸**：主要适配iPhone 13 (390x844)
- **设计风格**：简洁、现代、智能化

### 🎨 设计原则
1. **语音优先**：语音输入是核心功能，必须在界面中突出显示
2. **智能展示**：AI拆分的价值要清晰体现
3. **移动端友好**：支持单手操作，界面简洁直观
4. **功能聚焦**：核心功能突出，不堆砌复杂功能

### 🎨 必须设计的页面
1. **主页面**：语音输入中心布局
2. **录音界面**：录音状态和反馈
3. **处理界面**：AI处理流程展示
4. **结果界面**：AI拆分结果展示
5. **任务列表**：任务管理界面
6. **设置页面**：配置和偏好设置

## 🎨 具体页面设计要求

### 1. 主页面 (Home Screen)
**设计重点**：语音输入中心布局

**核心元素**：
- 大型录音按钮（视觉焦点）
- 今日任务预览（3-5个任务）
- 语音使用提示（具体示例）
- 最近添加的任务（快速访问）

**布局结构**：
```
顶部：标题栏 (TaskAI + 日期 + 天气 + 设置)
中间：录音区域 (大型按钮 + 使用提示)
下方：今日任务列表 (统计 + 搜索 + 筛选)
底部：快速操作按钮 (快速添加 + 统计)
```

**设计风格**：
- 录音按钮：渐变色 #667eea → #764ba2，120x120px
- 任务卡片：白色背景，圆角8px，轻微阴影
- 整体风格：简洁、现代、智能化

### 2. 录音界面 (Recording Screen)
**设计重点**：录音状态和反馈

**核心元素**：
- 录音状态指示（录音中、处理中、完成）
- 音频波形可视化
- 录音时长显示
- 控制按钮（暂停、停止、删除）
- 使用提示（支持中英文，最长60秒）

**设计风格**：
- 深色主题，突出录音状态
- 动态音频波形显示
- 清晰的控制按钮设计

### 3. 处理界面 (Processing Screen)
**设计重点**：AI处理流程展示

**核心元素**：
- 处理步骤（语音识别 → AI分析 → 任务拆分 → 完成）
- 进度条加载动画
- 处理时间预估
- 状态指示器

**设计风格**：
- 深色主题
- 清晰的步骤指示
- 流畅的加载动画

### 4. 结果界面 (Result Screen)
**设计重点**：AI拆分结果展示

**核心元素**：
- 原文识别文本
- AI拆分后的任务列表
- 编辑和修改功能
- 保存或重新录音选项

**设计风格**：
- 清晰的信息层级
- 任务项的可编辑设计
- 明确的操作按钮

### 5. 任务列表 (Task List)
**设计重点**：任务管理功能

**核心元素**：
- 任务分类（今天、明天、本周、本月）
- 搜索和筛选功能
- 任务状态管理（完成/未完成）
- 优先级和标签系统

**设计风格**：
- 清晰的分类标签
- 直观的状态指示
- 流畅的列表操作

### 6. 设置页面 (Settings)
**设计重点**：配置和偏好设置

**核心元素**：
- API配置状态
- 用户偏好设置
- 使用统计信息
- 帮助和反馈

**设计风格**：
- 清晰的设置分组
- 直观的状态指示
- 简洁的界面布局

## 🎨 设计系统要求

### 🎨 颜色系统
```
主色调：
- Primary: #667eea (语音按钮、强调元素)
- Secondary: #764ba2 (渐变补充)
- Background: #f8f9fa (页面背景)
- White: #ffffff (卡片背景)

功能色：
- Success: #4caf50 (完成任务)
- Warning: #ff9800 (进行中任务)
- Error: #f44336 (删除操作)
- Info: #2196f3 (信息提示)

中性色：
- Text Primary: #212121 (主要文字)
- Text Secondary: #757575 (次要文字)
- Text Tertiary: #b0b0b0 (辅助文字)
- Border: #e0e0e0 (边框线)
```

### 📝 字体系统
```
字体选择：系统默认字体（SF Pro Display / Roboto）
字体大小：
- Title: 22px (页面标题)
- Subtitle: 18px (副标题)
- Body: 16px (正文、列表项)
- Caption: 14px (标签、说明文字)
- Small: 12px (时间戳、状态提示)
```

### 🔧 组件系统
```
按钮：
- Primary Button: 背景色 #667eea，文字白色，圆角8px，高度48px
- Secondary Button: 背景#f5f5f5，文字#212121，圆角8px，高度44px
- Floating Button: 直径56px，背景色#667eea，白色图标

输入框：
- Height: 48px
- Border Radius: 8px
- Border: 1px solid #e0e0e0
- Focus Border: 2px solid #667eea

卡片：
- Background: #ffffff
- Border Radius: 12px
- Shadow: 0 2px 8px rgba(0,0,0,0.1)
- Padding: 16px
- Margin: 8px
```

## 🎨 交互设计要求

### 🎤 录音交互
- **开始录音**：按钮点击放大效果，进入录音状态
- **录音中**：音频波形动画，脉冲效果
- **处理中**：旋转加载动画，进度条指示
- **完成**：成功动画，自动跳转

### 📋 任务操作
- **点击任务**：展开详情页，显示编辑选项
- **滑动操作**：右滑完成，左滑删除或编辑
- **长按任务**：进入批量选择模式
- **拖拽排序**：调整任务顺序

### 🔍 搜索功能
- **实时搜索**：输入时实时过滤
- **历史记录**：显示最近搜索词
- **快捷筛选**：快速切换筛选条件

## 🎨 技术约束条件

### 📱 设备约束
- **主要设备**：iPhone 13 (390x844)
- **最小触摸区域**：44x44px
- **单手操作**：重要操作在拇指易达区域
- **安全区域**：考虑刘海屏和底部指示器

### 🎨 技术实现
- **技术栈**：React Native (推荐) 或 Flutter
- **API集成**：科大讯飞语音识别 + 百度千帆大模型
- **本地存储**：SQLite数据库
- **云端同步**：Firebase / AWS

### ⚡ 性能要求
- **启动时间**：< 3秒
- **响应时间**：< 2秒
- **流畅度**：60fps滚动和动画
- **内存占用**：< 100MB
