## 🔐 安全使用原则

### ✅ 正确做法
1. **仅在开发时使用真实密钥**
2. **不要在公共场合展示密钥**
3. **定期检查是否有密钥泄露**
4. **使用环境变量存储密钥（推荐）**

### ❌ 禁止行为
1. **不要将密钥提交到公开仓库**
2. **不要在社交媒体分享密钥**
3. **不要将密钥硬编码在客户端代码中**
4. **不要将密钥发送给不信任的人**

## 🛡️ 推荐的安全实践

### 1. 环境变量方式
```bash
# 设置环境变量
export XFYUN_APP_ID="your_app_id"
export XFYUN_API_KEY="your_api_key"
export XFYUN_API_SECRET="your_api_secret"
export BAIDU_API_KEY="your_api_key"
export BAIDU_SECRET_KEY="your_secret_key"
```

### 2. 配置文件方式
创建 `.env` 文件（不要提交到Git）：
```
XFYUN_APP_ID=your_app_id
XFYUN_API_KEY=your_api_key
XFYUN_API_SECRET=your_api_secret
BAIDU_API_KEY=your_api_key
BAIDU_SECRET_KEY=your_secret_key
```

在 `.gitignore` 中添加：
```
.env
*.env
config/production.js
```

### 3. 移动端安全存储
- **iOS**: 使用 Keychain Services
- **Android**: 使用 Keystore 或 Android Keystore
- **React Native**: 使用 react-native-keychain

## 🆘 如果发现密钥泄露

### 立即行动
1. **立即停用泄露的密钥**
2. **生成新的API密钥**
3. **更新所有使用密钥的代码**
4. **检查API调用日志**

### 各平台停用密钥
- **科大讯飞**：控制台 → 应用管理 → 停用服务
- **百度智能云**：控制台 → 应用管理 → 删除密钥
