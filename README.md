# TaskAI 项目

## 🎯 项目简介
TaskAI是一款基于语音识别和AI大模型的智能待办清单应用。用户可以通过语音输入任务描述，系统自动将语音转换为文字，并使用AI智能拆分为具体的可执行任务列表。

## 📋 项目状态
- **当前阶段**：技术验证完成 ✅
- **下一步**：移动端App开发
- **项目成功率**：85%+
